# 2026학년도 1학기 2차고사 시험 일정 시간표 웹앱

학교 전자칠판에 띄워 두고 현재 시간에 따라 시험 준비 시간, 시험 시간, 쉬는 시간, 자율학습, 시험 종료 상태를 안내하는 정적 웹앱입니다.

## 파일 설명

- `index.html`: 화면 구조와 기본 요소
- `style.css`: 전자칠판용 어두운 화면 디자인과 안전 영역 레이아웃
- `script.js`: 날짜/학년별 과목 데이터, 시간 계산, 구간 판정, 화면 갱신 로직
- `README.md`: 실행 방법, 수정 방법, 테스트 체크리스트

## 로컬에서 실행하는 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 날짜와 학년을 선택합니다.
3. 필요한 경우 테스트 시간을 입력합니다.
4. `시작` 버튼을 누릅니다.

별도 설치나 빌드 과정은 없습니다. HTML, CSS, JavaScript만 사용하므로 GitHub Pages에 올릴 수 있는 정적 파일 구조입니다.

## 테스트 시간 사용법

시작 화면의 테스트 시간 입력창에 `09:40` 또는 `09:40:30`처럼 입력하면 실제 현재 시각 대신 해당 시각을 기준으로 화면이 계산됩니다.

URL 파라미터도 사용할 수 있습니다.

```text
index.html?test=09:40
```

테스트 시간이 적용 중이면 진행 화면에 `테스트 모드`가 표시됩니다. 진행 화면에서는 테스트 시간을 직접 바꿀 수 없으며, `이전 화면으로` 버튼을 눌러 시작 화면에서 변경합니다.

## 전체화면 버튼 사용법

- 홈 화면과 시험 진행 화면 오른쪽 상단에 `전체화면` 버튼이 있습니다.
- 어느 화면에서든 `전체화면` 버튼을 누르면 브라우저 전체화면으로 전환됩니다.
- 전체화면 상태에서는 버튼 문구가 `전체화면 해제`로 바뀝니다.
- `전체화면 해제` 버튼을 누르거나 `Esc` 키를 누르면 전체화면을 해제할 수 있습니다.
- 홈 화면에서 시험 진행 화면으로 이동하거나 시험 진행 화면에서 홈 화면으로 돌아와도 버튼 문구는 현재 전체화면 상태와 일치합니다.
- 브라우저나 기기 정책에 따라 전체화면 전환이 제한될 수 있습니다. 이 경우 앱의 시험 시간표 기능은 그대로 유지됩니다.

## 시간표 수정 방법

`script.js`의 `EXAM_DAYS` 상수에서 날짜, 학년, 과목을 수정합니다.

```javascript
const EXAM_DAYS = {
  "2026-07-06": {
    label: "7월 6일 월요일",
    grades: {
      "2": ["자율", "기술·가정", "한문/생활일본어"],
      "3": ["기술·가정", "자율", "사회"],
    },
  },
};
```

`자율`은 시험 과목이 아니라 자율학습으로 처리됩니다. 화면에는 `자율학습`으로 표시되고, 시험 종료 10분 전 빨간색 강조가 적용되지 않습니다.

## 시간 구조 수정 방법

`script.js`의 `PERIODS` 상수를 수정합니다.

```javascript
const PERIODS = [
  {
    period: 1,
    prepStart: "09:00",
    prepEnd: "09:05",
    examStart: "09:05",
    examEnd: "09:50",
  },
];
```

쉬는 시간은 `BREAKS` 상수에서 수정합니다. 각 시간은 `HH:MM` 형식을 사용합니다.

## GitHub Pages 배포 방법 개요

1. GitHub 저장소를 만듭니다.
2. `index.html`, `style.css`, `script.js`, `README.md`를 저장소 루트에 올립니다.
3. 저장소의 `Settings`에서 `Pages` 메뉴로 이동합니다.
4. 배포 소스를 `Deploy from a branch`로 선택합니다.
5. 브랜치와 루트 폴더를 선택한 뒤 저장합니다.
6. GitHub Pages 주소가 생성되면 브라우저에서 접속해 동작을 확인합니다.

## 테스트 체크리스트

아래 시간은 시작 화면의 테스트 시간 입력창이나 `?test=시간` URL 파라미터로 확인합니다.

- `08:40` → `1교시 준비까지`
- `09:02` → `1교시 준비 시간`
- `09:06` → 1교시 시험 시간 또는 자율학습
- `09:39` → 시험 중, 남은 시간 기본색
- `09:40` → 시험 종료 10분 전, 시험 과목이면 남은 시간 빨간색
- `09:49` → 시험 종료 직전, 시험 과목이면 남은 시간 빨간색
- `09:50` → 쉬는 시간, 다음 시간이 자율학습이면 `다음 시간` / `자율학습` 두 줄 표시
- `10:06` → 2교시 준비 시간
- `10:46` → 2교시 시험 종료 10분 전, 시험 과목이면 남은 시간 빨간색
- `10:55` → 쉬는 시간, 다음 시간이 시험 과목이면 `다음 과목` / 과목명 두 줄 표시
- `11:12` → 3교시 준비 시간
- `11:50` → 3교시 시험 종료 10분 전, 시험 과목이면 남은 시간 빨간색
- `12:00` → `오늘 시험 종료`
- 자율학습 시간에는 빨간색 강조가 절대 적용되지 않는지 확인
- 쉬는 시간과 준비 시간에는 빨간색 강조가 절대 적용되지 않는지 확인
- 오른쪽 시간표에서 현재 교시가 정확히 강조되는지 확인
- 날짜/학년 선택을 바꾸었을 때 과목표가 정확히 바뀌는지 확인
- 홈 화면과 진행 화면 오른쪽 상단에 `전체화면` 버튼이 보이는지 확인
- 전체화면 전환 후 버튼 문구가 `전체화면 해제`로 바뀌고, 해제 후 다시 `전체화면`으로 돌아오는지 확인

## 사용자 직접 점검 절차

전자칠판에 실제로 띄우기 전 아래 순서대로 확인합니다.

1. 브라우저에서 `index.html`을 엽니다.
2. 시작 화면 제목 `2026학년도 1학기 2차고사`가 보이는지 확인합니다.
3. 날짜와 학년을 하나씩 선택했을 때 선택한 버튼의 테두리와 배경이 바뀌는지 확인합니다.
4. 테스트 시간에 `09:40`을 입력하고 `7월 6일 월요일`, `3학년`을 선택한 뒤 시작합니다.
5. 중앙 문구가 `기술·가정 시험 중`이고 남은 시간 숫자가 빨간색인지 확인합니다.
6. `이전 화면으로`를 누른 뒤 테스트 시간에 `09:40`을 입력하고 `7월 6일 월요일`, `2학년`을 선택한 뒤 시작합니다.
7. 중앙 문구가 `자율학습`이고 남은 시간 숫자가 빨간색이 아닌지 확인합니다.
8. 테스트 시간에 `10:05`를 넣고 `7월 6일 월요일`, `3학년`을 선택한 뒤 중앙 문구가 `자율학습 준비 시간`인지 확인합니다.
9. 테스트 시간에 `08:40`을 넣고 중앙 문구가 `1교시 준비까지`로 짧게 표시되는지 확인합니다.
10. 테스트 시간에 `09:50`을 넣고 쉬는 시간 문구가 `다음 시간` / `자율학습` 또는 `다음 과목` / 과목명처럼 두 줄로 표시되는지 확인합니다.
11. 테스트 시간에 `10:55`를 넣고 긴 과목명도 단어 중간이 어색하게 끊기지 않는지 확인합니다.
12. 오른쪽 시간표가 화면 오른쪽 끝에 붙지 않고, 화면 안쪽에 여유를 두고 표시되는지 확인합니다.
13. 전자칠판에서 화면 위쪽, 왼쪽, 오른쪽 끝, 아래쪽이 가려져도 중앙 문구와 남은 시간, 오른쪽 시간표가 읽히는지 확인합니다.
14. 홈 화면 오른쪽 상단의 `전체화면` 버튼이 제목, 날짜 선택, 학년 선택, 시작 버튼과 겹치지 않는지 확인합니다.
15. 홈 화면에서 `전체화면` 버튼을 눌러 전체화면으로 전환한 뒤 버튼 문구가 `전체화면 해제`로 바뀌는지 확인합니다.
16. 전체화면 상태에서 시험 진행 화면으로 이동해도 오른쪽 상단 버튼 문구가 `전체화면 해제`로 유지되는지 확인합니다.
17. 진행 화면 오른쪽 상단의 `전체화면 해제` 버튼이 날짜·학년 표시와 겹치지 않는지 확인합니다.
18. `전체화면 해제` 버튼이나 `Esc` 키로 전체화면을 해제한 뒤 홈 화면과 진행 화면의 버튼 문구가 다시 `전체화면`으로 돌아오는지 확인합니다.
19. 진행 화면에서 다시 `전체화면` 버튼을 눌러 전체화면으로 전환한 뒤 `이전 화면으로`를 눌러도 홈 화면 버튼 문구가 현재 전체화면 상태와 일치하는지 확인합니다.
20. `?test=09:40`처럼 URL 뒤에 테스트 시간을 붙여도 같은 방식으로 동작하는지 확인합니다.
